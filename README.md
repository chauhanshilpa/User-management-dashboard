# User Management Dashboard

A React + TypeScript app for browsing and managing users via the [DummyJSON](https://dummyjson.com) API.

## Features

- **User list** — paginated table showing avatar, name, email, phone, company, and role badge
- **Search** — filter users by name or email in real time
- **Filters** — narrow results by role or gender
- **Sorting** — sort by name (A–Z / Z–A) or age (low–high / high–low)
- **Add user** — create a new user via a modal form
- **Edit user** — update an existing user's details
- **Delete user** — remove a user with a confirmation dialog
- **User detail page** — full profile view with personal, address, company, and additional info
- **Skeleton loading** — placeholder rows shown while data is fetching
- **Toast notifications** — brief success/error messages after actions

## Stack

- React 18, TypeScript, Vite
- React Router v6
- DummyJSON REST API
